# Gviz

Ruby's interface of graphviz. It generate a dot file with simple ruby's syntax.

## Installation

Add this line to your application's Gemfile:

    gem 'gviz'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install gviz
